# 🚀 Chandu Pulluru Portfolio

This is my personal portfolio website built using **Next.js** and **Tailwind CSS**.

## 🔥 Features

* Fully responsive (mobile + desktop)
* SEO optimized using metadata
* Fast performance with optimized images
* Smooth animations using Framer Motion

## 🛠️ Tech Stack

* Next.js
* Tailwind CSS
* Framer Motion

## 📂 Sections

* Hero (Name & Role)
* About
* Skills
* Projects
* Contact

## ⚡ Performance

* Optimized images using Next.js Image
* Minimal dependencies
* Clean code
